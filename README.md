# SetHome & Warp Mod for Better Than Wolves Community Edition (3.x)

This mod adds comprehensive home and warp commands to Better Than Wolves Community Edition, making world navigation and multiplayer interaction more convenient and flexible.

---

## Features

- **Set and Delete Homes:** `/sethome [name]` and `/delhome [name]` to save or remove personal home points.
- **List Homes:** `/listhome` displays all your saved home points.
- **Teleport to Home:** `/home [name]` teleports you to a saved home point, with cooldown and dimension checks.
- **Share Homes:** `/sharehome [name]` broadcasts your home location to all players.
- **Warp Points:** `/setwarp [name]`, `/delwarp [name]`, `/listwarp`, `/warp [name]` for server-wide warp management.
- **Share Warps:** `/sharewarp [name]` broadcasts a warp location to all players.
- **Player Teleport Requests:** `/t <player>`, `/t yes`, `/t no` for convenient player-to-player teleportation with request/accept/deny mechanics.

---

## Command Reference

- `/sethome [name]` — Set your current location as a home point.
- `/delhome [name]` — Delete a saved home point.
- `/listhome` — List all your home points.
- `/home [name]` — Teleport to a home point (with a 5-second delay).
- `/sharehome [name]` — Share your home location with all players.
- `/setwarp [name]` — Set a global warp point.
- `/delwarp [name]` — Delete a global warp point.
- `/listwarp` — List all global warp points.
- `/warp [name]` — Teleport to a warp point (with a 5-second delay).
- `/sharewarp [name]` — Share a warp location with all players.
- `/t <player>` — Request to teleport to another player(with a 5-second delay).
- `/t yes` — Accept a teleport request.
- `/t no` — Deny a teleport request.

## Requirements

- Minecraft with [Better Than Wolves Community Edition 3.x]()

---

## How to Install

1. [Download the latest release (.jar)](https://github.com/Inf1nlty/SetHome-BTWCE3/releases/latest)
2. Place it into your `mods` folder.
3. Start Minecraft with BTW Community Edition.

---

## Notes

- Home and warp points are dimension-sensitive; you cannot teleport across dimensions.
- All commands are accessible to all players by default.
